## Todo

A simple todo application with typical old user password login.

docker image link: https://hub.docker.com/r/scewdoosh/spring_todo_backend/tags <br>

download docker, pull image using `docker pull scewdoosh/spring_todo_backend:latest`

run the image using `docker run -d -p 8080:8080 skadoosh/spring-todo-backend:latest`

swagger endpoint : `/api/swagger`
